## Architecture Overview

### Backend (Spring Boot 3.2.12 / Java 17)
- **Bounded Contexts:** Task Management (core), Identity & Access (supporting)
- **DDD Layers:** domain/model, domain/event, domain/repository, application/service, infrastructure/persistence, api/controller, api/dto
- **DDD Health Score:** 95/100 (validated by governance agent)
- **Key Patterns:** Aggregate roots, value objects, repository port/adapter, MapStruct DTO mapping

### Frontend (Angular 17.3)
- **Feature Modules:** today, tasks, lists, auth (all lazy-loaded)
- **Layouts:** MainLayout (sidebar + content), AuthLayout (centered card)
- **Core Services:** AuthService, TokenStorageService, auth guard, HTTP interceptors

### Database (PostgreSQL 15)
- **Migrations:** V1-V5 (utility functions, users, refresh_tokens, task_lists, tasks)
- **Schema:** UUID primary keys, audit columns, soft delete, proper foreign keys

### Infrastructure
- **Docker Compose:** PostgreSQL 15-alpine + Redis 7-alpine
- **CI/CD:** GitHub Actions (backend Maven verify + JaCoCo, frontend lint + test + build)
- **Security Scanning:** TruffleHog + OWASP Dependency-Check in CI
